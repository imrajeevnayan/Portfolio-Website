# Portfolio Design Style Guide

## Design Philosophy

### Color Palette
- **Primary**: Deep Navy (#1a1d29) - Professional, trustworthy
- **Secondary**: Warm Amber (#f59e0b) - Innovation, energy
- **Accent**: Soft Teal (#14b8a6) - Technology, reliability
- **Neutral**: Light Gray (#f8fafc) - Clean, modern
- **Text**: Charcoal (#374151) - Readable, sophisticated

### Typography
- **Display Font**: "Playfair Display" - Elegant serif for headings
- **Body Font**: "Inter" - Clean, modern sans-serif for content
- **Code Font**: "JetBrains Mono" - Technical documentation

### Visual Language
- **Minimalist Sophistication**: Clean lines, generous whitespace
- **Technical Precision**: Sharp edges, geometric patterns
- **Professional Warmth**: Balanced use of color and space
- **Modern Elegance**: Subtle shadows and refined interactions

## Visual Effects

### Used Libraries
- **Anime.js**: Smooth animations and transitions
- **ECharts.js**: Interactive skill visualizations
- **Pixi.js**: Background particle effects
- **Splide.js**: Project carousel
- **p5.js**: Creative coding elements

### Effects & Animations
- **Hero Background**: Subtle particle system with floating geometric shapes
- **Text Animations**: Staggered letter reveals and color cycling
- **Skill Bars**: Animated progress indicators with easing
- **Project Cards**: 3D hover effects with depth shadows
- **Timeline**: Smooth scroll-triggered animations

### Styling Approach
- **Grid-Based Layout**: Consistent spacing and alignment
- **Subtle Gradients**: Minimal use for depth and interest
- **Interactive States**: Smooth transitions on hover/focus
- **Responsive Design**: Mobile-first approach
- **Accessibility**: High contrast ratios and keyboard navigation

### Header Effect
- **Animated Background**: Flowing particles representing data streams
- **Typography Animation**: Name appears with letter-by-letter reveal
- **Skill Tags**: Floating technology badges with gentle motion
- **Call-to-Action**: Pulsing button with subtle glow effect
